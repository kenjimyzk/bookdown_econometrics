11.1 logit

11.1.1 数値計算

N <- 100
x <- rnorm(N, 1, 2)
ystar <- -1 + x + rlogis(N)
y <- ifelse(ystar >= 0, 1, 0)
df <- data.frame(x, y, ystar)
fm_logit <- glm(y ~ x, data = df, family = binomial)
summary(fm_logit)
#>

11.1.4 擬決定係数

11.1.5 限界効果
